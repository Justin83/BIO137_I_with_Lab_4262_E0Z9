# BIO 137 Blackboard Pages

This repository is for HTML pages and related snippets that need to be displayed inside Blackboard.

Current intended use:

- Store Blackboard-ready HTML pages.
- Keep page source under version control.
- Avoid using this repository as a general public course website.

## Folder structure

```text
pages/
  syllabus/
    syllabus.html
```

Older or experimental paths may remain temporarily until Blackboard links are confirmed.
